## What I Learned About GitHub

### Repositories
Repositories store project files and maintain version history.

### Issues
Issues help track bugs, tasks, and feature requests.

### Pull Requests
Pull requests allow code review and collaboration before merging changes.

### Branches
Branches help developers work independently without affecting the main codebase.

### Forks
Forks are personal copies of repositories that allow experimentation and contributions.

### GitHub Projects
GitHub Projects provide task management using boards and workflows.
